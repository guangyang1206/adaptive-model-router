# Code of Conduct

We want Adaptive Model Router to be a respectful and constructive open-source project.

## Expected behavior

- Be respectful and direct.
- Discuss ideas based on evidence and trade-offs.
- Assume good intent, but keep technical standards high.
- Welcome contributors across different model ecosystems and regions.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing private credentials, tokens, logs, or personal information.
- Abusive or intentionally disruptive behavior.

## Reporting

For now, please report issues through GitHub Issues once the public repository is available.
